# SCT_DS_1
# Top 10 Most Populous Countries in 2023
This project analyzes global population data and visualizes the **Top 10 Most Populous Countries in 2023** using a horizontal bar chart.
## Overview
The notebook/script:
1. Loads population data from a World Bank dataset.
2. Loads country metadata.
3. Filters countries that belong to valid regions.
4. Extracts population values for the year 2023.
5. Identifies the top 10 most populous countries.
6. Creates a horizontal bar chart for visualization.
## Dataset
The project uses:
- `API_SP.POP.TOTL_DS2_en_csv_v2_267553.csv` – Population dataset
- `Metadata_Country_API_SP.POP.TOTL_DS2_en_csv_v2_267553.csv` – Country metadata
These files can be downloaded from the World Bank Open Data portal.
